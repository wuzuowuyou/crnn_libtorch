# crnn_pytorch

For the Pytorch implementation, you can refer to [wuzuowuyou/CRNN_Chinese_Characters_res18_senet](https://github.com/wuzuowuyou/CRNN_Chinese_Characters_res18_senet)


## dependence
```
OpenCV >= 3.4
libtorch = 1.7.0
cuda10.2
```
本git在上述环境下没有问题，在libtorch其他版本应该也是可以的。
